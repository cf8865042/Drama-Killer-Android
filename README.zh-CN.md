# 短剧平台杀手 - 概述

## 什么是短剧平台杀手？

短剧平台杀手是一款专业的媒体缓存应用程序，旨在通过允许您缓存短剧平台上的媒体内容来增强您的观看体验，支持离线观看。

## 核心概念

**短剧平台杀手不直接提供媒体内容。** 相反，它作为一个缓存工具：

- 连接到现有的短剧平台
- 将媒体内容缓存到您的设备
- 支持离线观看缓存的内容
- 提供高效的文件管理和组织

## 工作原理

1. **内容发现**：应用嗅探短剧平台页面以识别可用的媒体内容
2. **离线访问**：一旦缓存，内容无需互联网连接即可离线观看
3. **文件管理**：将下载的内容组织成逻辑组以便轻松访问

## 主要功能

- **网页浏览器集成**：直接在应用内浏览短剧平台
- **自动下载**：自动检测和下载检测到的视频
- **文件分组**：将内容组织成逻辑组
- **暗模式支持**：网页暗模式，提供更好的观看体验
- **多语言支持**：支持英文和中文
- **下载管理**：监控和控制下载任务

## 使用场景

- **离线观看**：无需互联网连接观看您喜爱的短剧
- **内容备份**：创建重要媒体内容的本地备份
- **旅行娱乐**：在旅行前下载内容
- **节省数据**：避免重复流式传输相同内容

## 法律声明

本应用程序专为个人使用和内容备份目的而设计。用户有责任确保他们有权缓存和存储所下载的内容。请尊重版权法和平台服务条款。

## 系统要求

- Android 6.0（API 级别 23）或更高版本
- 足够的媒体内容存储空间
- 用于初始内容发现和缓存的互联网连接
